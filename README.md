# cypress-ecommerce-tests
Testes automatizados para e-commerce com Cypress
É um projeto de testes automatizados com Cypress para um fluxo de e-commerce.
Eu utilizei Page Object para organizar a automação e fixture para separar os dados de teste, como formas de pagamento.
Isso ajuda a deixar os testes mais limpos, reutilizáveis e fáceis de manter.
